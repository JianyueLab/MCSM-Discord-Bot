## MCSM-Discord-Bot （开发中）

### Introduction:

This bot is based on MCSManager Panel, and control instance on discord.

本bot是基于[MCSManager](https://mcsmanager.com)的面板，所创建的discord控制机器人。

### Features:

- Get status and Instances
- Check specific instances
- Start and stop Instances

- 获取节点和实例的ID
- 获取实例的状态
- 控制实例的开关
